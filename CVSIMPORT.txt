These are notes pertaining to 'cvs import' commands required to maintain
the GraphicsMagick delegate libraries in GraphicsMagick CVS. As such, it
is intended for developers only.

The commands are only examples and must be modified based on the
CVS user name and package version.

bzlib:

  cvs -d :ext:bfriesen@cvs.graphicsmagick.org:/GraphicsMagick \
    import -m 'Import bzip2 1.0.2' delegates/bzlib \
    INITIAL BZIP2_1_0_2

FreeType:
cvs -d :ext:bfriesen@cvs.graphicsmagick.org:/GraphicsMagick \
    import -m 'Import freetype 2.1.5' delegates/freetype2 \
    FREETYPE2 FREETYPE_2_1_5

jasper
  cvs -d :ext:bfriesen@cvs.graphicsmagick.org:/GraphicsMagick \
    import -m 'Import jasper 1.700.2' delegates/jp2 \
    JASPER_DIST JASPER_1_700_2

jbig:

  cvs -d :ext:bfriesen@cvs.graphicsmagick.org:/GraphicsMagick \
    import -m 'Import jbigkit 1.5' delegates/jbig \
    INITIAL JBIGKIT_1_5

lcms:
  
  cvs -d :ext:bfriesen@cvs.graphicsmagick.org:/GraphicsMagick \
    import -m 'Import lcms 1.14' delegates/lcms \
    LCMS_DIST LCMS_1_14
  cvs -d :ext:bfriesen@cvs.graphicsmagick.org:/GraphicsMagick \
    checkout -j LCMS_1_13 -jLCMS_1_14 delegates/lcms

png:

  cvs -d :ext:bfriesen@cvs.graphicsmagick.org:/GraphicsMagick \
    import -m 'Import libpng  1.2.7' delegates/png \
    INITIAL LIBPNG_1_2_7

ralcgm:
  cvs -d :ext:bfriesen@cvs.graphicsmagick.org:/GraphicsMagick \
    import -m 'Import ralcgm 3.50' delegates/ralcgm \
    RALCGM_DIST RALCGM_3_50

wmf:

  cvs -d :ext:bfriesen@cvs.graphicsmagick.org:/GraphicsMagick \
    import -m 'Import CVS libwmf 0.2.8.2' delegates/wmf \
    LIBWMF_DIST LIBWMF_0_2_8_2

  Where 'LIBWMF_DIST' is the CVS vendor branch, and 'LIBWMF_0_2_8_2' is the
  vendor release tag.
